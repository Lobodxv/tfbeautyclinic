# 🌹 TF Beauty Clinic - Website

Um site moderno, responsivo e informacional para a clínica de estética **TF Beauty Clinic**, desenvolvido com HTML5, CSS3 e JavaScript puro (Vanilla JS).

## 🎯 Características

- ✅ **Design Responsivo** - Otimizado para mobile-first (prioriza dispositivos móveis)
- ✅ **Tema Moderno** - Cores elegantes: branco e vermelho bordô (#8B1538)
- ✅ **Navegação Intuitiva** - Menu responsivo e smooth scrolling
- ✅ **Seções Completas**:
  - Hero section atrativa
  - Sobre a profissional/clínica
  - Galeria de procedimentos com antes/depois
  - Formulário de contato
  - Informações de contato
- ✅ **Interatividade**:
  - Menu mobile com toggle
  - Comparador de antes/depois (arrastar/toque)
  - Animações no scroll
  - Validação de formulário
- ✅ **Totalmente Acessível** - Suporte para leitores de tela e teclado
- ✅ **Performance Otimizada** - Carregamento rápido e eficiente

## 📱 Responsividade

O site é otimizado para:
- **Móvel** (até 480px) - Prioridade máxima
- **Tablet** (481px - 768px)
- **Desktop** (acima de 768px)

### Breakpoints:
```css
/* Mobile */
max-width: 480px

/* Tablet */
max-width: 768px

/* Desktop */
Sem limite máximo
```

## 🔧 Funcionalidades JavaScript

### 1. **Menu Responsivo**
   - Toggle do menu em dispositivos móveis
   - Animação de hambúrguer
   - Fechar ao clicar em um link

### 2. **Comparador Antes/Depois**
   - Arraste o slider com o mouse (desktop)
   - Toque e deslize (mobile)
   - Comparação visual de resultados

### 3. **Smooth Scrolling**
   - Navegação suave entre seções
   - Links internos funcionam perfeitamente

### 4. **Animações no Scroll**
   - Cards aparecem com animação ao scroll
   - Efeito fade-in gradual

### 5. **Validação de Formulário**
   - Verifica campos obrigatórios
   - Mostra mensagem de sucesso ao enviar
   - Feedback visual de erros
